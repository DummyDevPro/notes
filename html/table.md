# <table> Element
## Simple HTML Table
```html
<body>
    <table border="1px" cellspacing="0">
        <tr>
            <th>Name</th>
            <th>Country</th>
        </tr>
        <tr>
            <td>KUMAR</td>
            <td>Nepal</td>
        </tr>
        <tr>
            <td>ANGEL</td>
            <td>Puerto Rico</td>
        </tr>
        <tr>
            <td>PARIK</td>
            <td>Nepal</td>
        </tr>
    </table>
</body>
```
### Output
| Name   | Country     |
| ------ | -------     |
| KUMAR  | Nepal       |
| ANGEL  | Puerto Rico |
| PARIK  | Nepal       |
